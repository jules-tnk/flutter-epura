DATA LAYER:
┌────────────────────────────────────────────┐
│  Provider                                  │
│  ├── FileService (scan photos/videos/dl)   │
│  ├── ReviewProvider (swipe state, queue)    │
│  ├── StatsProvider (history, aggregation)   │
│  └── SettingsProvider (prefs, reminder)     │
│                                            │
│  Local Storage: SharedPreferences + SQLite │
│  File Access: photo_manager + path_provider│
│  Notifications: flutter_local_notifications│
└────────────────────────────────────────────┘

Key Packages

provider — state management
photo_manager — access photos/videos
flutter_local_notifications — reminders
sqflite — review history DB
shared_preferences — settings
path_provider — downloads folder
fl_chart — stats charts
flutter_card_swiper — swipe cards
